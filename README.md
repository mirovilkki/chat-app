Chat app is POC for FRP javascript app with websockets.

Frontend Inludes:
- React
- Kefir
- Ramda
- kefir.atom
- karet.utils
- karet
- partial.lenses
- ES6

Backend includes:
- Node
- Express
- Socket.io

To run localy check deps and in both folders frontend and backend run `npm i` and `npm start`.

Env deps:
- nodemon v1.11.0
- node v7.10.0
- npm v4.2.0
